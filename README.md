# typesafeconfig

TypeScript base configuration (tsconfig) to extend from with emphasis on type safety.

## Installation

```shell
npm install --save-dev typesafeconfig
```

## Usage

In your `tsconfig.json`:

```jsonc
{
  "extends": "typesafeconfig",
}
```
